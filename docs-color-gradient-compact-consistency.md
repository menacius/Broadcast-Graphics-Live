# Color / Gradient Popup Compact Consistency

- Rebalanced the Color tab to be more compact and consistent with the Gradient tab proportions.
- Reduced the HSV color wheel footprint and tightened panel margins, spacing, swatches, recent colors, and numeric controls.
- Reused the existing sidebar/canvas eyedropper workflow from the Color tab instead of launching a second picker dialog.
- Tightened the Gradient tab to match the Color tab scale: smaller preview, smaller stop controls, tighter spacing, and smaller preset chips.
