# Sidebar flyout markers, swatches, and shortcuts

This pass intentionally keeps the original one-column sidebar dimensions and layout.

- Flyout-capable tool buttons paint a small filled bottom-right corner caret.
- Foreground/background swatches retain their square overlapping geometry and use a dark outer plus light inner border for contrast.
- Illustrator-style shortcuts are handled by the editor only when no text/value editor has focus: V Selection, M Rectangle, L Ellipse, T Text, I Eyedropper, G Gradient.
