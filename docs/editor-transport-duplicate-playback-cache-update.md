# Editor transport, duplication, and Playback and Cache update

- Added Ctrl+D layer duplication through the existing multi-layer clone and naming pipeline.
- Removed the Loop Preview transport button.
- Added Go to Start, Step Backward, reverse playback with a mirrored Play icon, and Go to End controls.
- Reordered transport controls around reverse/forward playback and corrected the Play tooltip.
- Added a distinct Playback Mode option to the prerender Mode selector. It follows the title/graphic playback configuration, while Loop, Ping-Pong Loop, and Play Once remain explicit preview overrides.
- Renamed the Prerender editor dock and its Window menu action to Playback and Cache.
