# Illustrator-style compact sidebar refinement

- Increased the sidebar width to 100 px while reducing tool buttons to 30×30 px and icons to 18×18 px.
- Kept tools in a two-column grid with DPI-safe margins and spacing so buttons and flyout carets are not clipped.
- Moved the fill/stroke control to the bottom of the sidebar, following Illustrator's toolbar hierarchy.
- Reworked foreground/background swatches as overlapping 38×38 px squares with light/dark double outlines for visibility on any fill.
- Reduced flyout caret geometry to fit the smaller buttons cleanly.
