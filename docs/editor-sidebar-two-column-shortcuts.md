# Editor sidebar two-column layout and tool shortcuts

The editor tool sidebar now uses a wider two-column grid. Tools that expose a flyout menu draw a compact filled lower-right caret without relying on the platform menu indicator. The fill/stroke swatch occupies the sidebar width and uses a light/dark double outline so solid colors, gradients, and transparency remain legible across OBS themes.

Initial Illustrator-compatible tool shortcuts:

- `V`: Selection
- `M`: Rectangle
- `T`: Text
- `I`: Eyedropper
- `G`: Gradient

Single-key tool shortcuts are ignored while focus is in a text editor, line edit, spin box, combo box, or another value-entry control.
