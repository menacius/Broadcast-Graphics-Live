# Gradient Tab Square Layout Refinement

- Reworked the Gradient tab to use a more square, compact layout closer to the Color mockup proportions.
- Moved the gradient preset strip to the bottom of the tab.
- Increased the gradient preview/ramp height so the tab reads as a primary editor panel instead of a compressed form.
- Simplified the visible gradient controls to keep the tab compact.
- Removed the custom per-stop HSV popup from the Gradient tab and routed stop color editing through the existing Qt/OBS color dialog instead of introducing another color picker implementation.
