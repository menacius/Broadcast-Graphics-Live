# Appearance Preferences: Timeline and Canvas

The Preferences dialog Appearance page is now split into two visual categories:

- Timeline
- Canvas

The Timeline category keeps the existing timeline color controls.

The Canvas category adds persistent color/opacity controls for editor-only helper elements, including:

- user guidelines
- active/dragged guidelines
- ruler mouse indicators
- hover bounding boxes
- selection bounding boxes
- text bounding boxes
- snap lines
- canvas border
- action safe guides
- graphics safe guides
- scene mask object outlines

All new canvas helper settings are stored under the `CanvasHelperColors` settings group and are applied only to editor chrome. They do not alter the title artwork or OBS source output.
