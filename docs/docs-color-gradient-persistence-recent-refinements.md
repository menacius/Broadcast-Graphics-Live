# Color / Gradient Persistence Refinements

- Reopens the Color tab at the same screen position after using the canvas/sidebar color picker from the Color tab.
- Keeps layer effects out of the new-object style/default persistence path.
- Stores new-object default styling with the saved title/template data, including foreground/background defaults.
- Adds recent color commit behavior that records a color only when the color popup is closed by the user, not when it closes temporarily for the color picker, and removes duplicate entries before inserting the latest color.
