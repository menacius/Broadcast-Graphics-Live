# Gradient Canvas/Tabbed Workflow Fix

- Canvas gradient handles are now visible only when the Gradient tabbed editor is active or when the sidebar Gradient Tool is selected.
- Fill/Stroke swatch clicks now open the existing tabbed selector on the appropriate tab: Color for solid fills/strokes and Gradient for gradient fills/strokes.
- Switching to the Color tab immediately converts/applies the selected color to the target fill/stroke, while switching to the Gradient tab applies the gradient state.
- The sidebar Gradient Tool can now edit an existing selected object's gradient handles instead of always starting a new gradient drag.
